# 🚀 Portfólio Pessoal Interativo

Projeto desenvolvido com HTML, CSS e JavaScript, criado para apresentar meu perfil profissional e facilitar o acesso às minhas redes e projetos.

## ✨ Funcionalidades
<p>- Página de apresentação pessoal.</p>
<p>- Exibição do perfil profissional.</p>
<p>- Links para GitHub e LinkedIn.</p>
<p>- Interações dinâmicas com JavaScript.</p>
<p>- Design estilizado com CSS.</p>
<p>- Layout responsivo para diferentes dispositivos.</p>
<p>- Estrutura simples para aprendizado em desenvolvimento web.</p>
