#Portfólio Pessoal Interativo

Projeto desenvolvido com HTML, CSS e JavaScript, criado para apresentar meu perfil profissional e facilitar o acesso às minhas redes e projetos.

#Funcionalidades
Página de apresentação pessoal
Exibição do perfil profissional
Links para GitHub e LinkedIn
Interações dinâmicas com JavaScript
Design estilizado com CSS
Layout responsivo para diferentes dispositivos
Estrutura simples para aprendizado em desenvolvimento web

#Tecnologias
HTML5
CSS3
JavaScript

# Objetivo do Projeto
Este projeto foi criado com o objetivo de praticar desenvolvimento Front-end, trabalhando estruturação de páginas, estilização e interatividade utilizando as principais tecnologias da web.
